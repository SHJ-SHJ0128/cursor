HTML/CSS 学习

什么是HTML和CSS？

HTML（超文本标记语言）是网页的基础结构，它定义了网页的内容和基本布局。CSS（层叠样式表）则负责控制网页的外观和样式，包括颜色、字体、布局等。这两种语言是前端开发的基础，掌握它们是成为前端开发者的第一步。

HTML基础

HTML是一种标记语言，使用各种标签来定义网页结构。以下是一些基本的HTML标签：

<!DOCTYPE html>
<html>
<head>
  <title>我的第一个网页</title>
</head>
<body>
  <h1>这是一个标题</h1>
  <p>这是一个段落。</p>
</body>
</html>

这是一个基本的HTML文档结构，包含了头部和正文内容。

CSS基础

CSS用于为HTML元素添加样式。CSS可以通过多种方式应用到HTML中：

/* 内联样式 */
<p style="color: blue;">这是蓝色文本</p>

/* 内部样式表 */
<style>
  p {
    color: red;
    font-size: 16px;
  }
</style>

/* 外部样式表 */
<link rel="stylesheet" href="styles.css">

CSS使用选择器定位HTML元素，然后应用样式规则。

布局技术

现代CSS提供了多种布局技术，包括：

  • Flexbox布局
  • Grid网格布局
  • 响应式布局
  • 盒模型
/* Flexbox示例 */
.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

Flexbox和Grid是现代网页布局的主要工具，它们提供了强大且灵活的布局能力。
