JavaScript 学习

什么是JavaScript？

JavaScript是一种轻量级的、解释型的、面向对象的编程语言，是网页交互的核心技术。它可以使网页动态响应用户操作，实现复杂的交互功能。如今，JavaScript不仅可以运行在浏览器中，还可以通过Node.js在服务器端运行，成为了全栈开发的关键语言。

JavaScript基础

JavaScript的基础语法包括变量、数据类型、运算符和控制流等：

// 变量声明
let name = 'JavaScript学习';
const PI = 3.14159;
var oldWay = '旧的声明方式';

// 数据类型
let num = 100; // 数字
let str = "Hello"; // 字符串
let bool = true; // 布尔值
let arr = [1, 2, 3]; // 数组
let obj = {name: "小明"}; // 对象
let nothing = null; // 空值
let undef; // undefined

了解这些基础概念是掌握JavaScript的第一步。

函数与事件

JavaScript中的函数可以以多种方式定义，它们是代码复用和事件处理的关键：

// 函数声明
function sayHello(name) {
  return `你好，${name}！`;
}

// 箭头函数
const multiply = (a, b) => a * b;

// 事件处理
document.getElementById('myButton').addEventListener('click', function() {
  alert('按钮被点击了！');
});

函数是JavaScript中的一等公民，它们可以作为参数传递，也可以作为返回值。

DOM操作

DOM（文档对象模型）操作是JavaScript与网页交互的核心：

// 获取元素
const title = document.getElementById('title');
const paragraphs = document.querySelectorAll('p');

// 修改内容和样式
title.innerHTML = '新标题';
title.style.color = 'blue';

// 创建和添加元素
const newDiv = document.createElement('div');
newDiv.textContent = '这是新创建的元素';
document.body.appendChild(newDiv);

通过DOM操作，可以动态地修改网页内容和样式，实现交互效果。
